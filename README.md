# NNPTUD_BUOI3 - Product Dashboard

## Mô tả
Dashboard hiển thị danh sách sản phẩm từ API với các tính năng tìm kiếm, sắp xếp, phân trang và tooltip.

## Các tính năng
1. ✅ Lấy dữ liệu từ API: https://api.escuelajs.co/api/v1/products
2. ✅ Hiển thị bảng với dòng xen kẽ (đen và trắng)
3. ✅ Tìm kiếm sản phẩm theo tiêu đề (real-time)
4. ✅ Phân trang với 5, 10 hoặc 20 sản phẩm/trang
5. ✅ Sắp xếp theo tên và giá (tăng/giảm)
6. ✅ Ẩn cột description, hiển thị khi hover
